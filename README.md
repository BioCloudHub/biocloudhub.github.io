# BioCloudHub 首页

这是 BioCloudHub 的专业生物制药行业首页，基于 Vite + Vue 3 + Tailwind CSS 构建。

## 特色
- **专业视觉设计**：采用生物医药行业的高级色调（深蓝、青绿色），精致且具有科技感。
- **响应式布局**：完美适配桌面、平板与移动端。
- **行业聚焦内容**：涵盖研发、临床、制造与合规四大核心板块。
- **高性能**：基于 Vite 构建，极速加载。

## 快速开始

1. **安装依赖**
   ```bash
   npm install
   ```

2. **启动开发服务器**
   ```bash
   npm run dev
   ```

3. **构建生产版本**
   ```bash
   npm run build
   ```

## 目录结构
- `src/App.vue`: 首页核心组件。
- `src/style.css`: 全局样式与 Tailwind 指令。
- `tailwind.config.js`: 主题与色彩自定义。
